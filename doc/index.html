• OmegaEngine API documents the public interface provided by the engine, including the OmegaGUI toolkit. Intended for developers building apps with the engine.
  • AlphaFramework API documents the public interface provided by the framework, including the AlphaEditor. Intended for developers building games with the framework.
  • Frame of Reference Game API documents the internal structure of the game. Intended as a sample/reference for developers working on other game.
